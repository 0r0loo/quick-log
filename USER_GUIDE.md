# Quick Log 사용자 가이드

## 소개

Quick Log는 자바스크립트/타입스크립트 개발자를 위한 JetBrains IDE 플러그인으로, 디버깅을 위한 console.log 문을 빠르고 효율적으로 삽입할 수 있게 해줍니다.

## 주요 기능

### 1. 스타일링된 console.log 삽입

Quick Log는 다음과 같은 형식의 console.log 문을 자동으로 생성합니다:

```javascript
console.log('파일명:라인번호 | 변수명 : ', 변수값);
```

이 형식은 디버깅 시 로그의 출처와 변수 정보를 명확하게 파악할 수 있도록 돕습니다.

### 2. 멀티 커서 지원

변수를 선택하지 않고 Quick Log 단축키를 누르면, 멀티 커서 기능이 활성화된 템플릿 로그가 삽입됩니다:

```javascript
console.log('파일명:라인번호 | ', );
```

커서가 자동으로 배치되어 라벨과 값을 동시에 입력할 수 있습니다.

### 3. 들여쓰기 자동 유지

삽입된 console.log 문은 현재 코드의 들여쓰기 레벨을 자동으로 유지합니다.

## 사용 방법

### 기본 사용법

1. 코드에서 로깅하고 싶은 변수나 표현식을 선택합니다.
2. `Alt+L` 단축키를 누릅니다.
3. 선택한 변수에 대한 console.log 문이 현재 라인 다음에 삽입됩니다.

### 선택 없이 사용하기

1. 로그를 삽입하고 싶은 위치에 커서를 놓습니다.
2. `Alt+L` 단축키를 누릅니다.
3. 템플릿 console.log 문이 삽입되고, 라벨과 값 입력 위치에 멀티 커서가 활성화됩니다.

## 단축키 설정 변경

1. JetBrains IDE에서 `Settings/Preferences`를 엽니다.
2. `Tools` > `Quick Log Settings`로 이동합니다.
3. 원하는 단축키를 입력합니다 (예: `ctrl shift L`).
4. 변경 사항을 적용하려면 IDE를 재시작해야 합니다.

## 팁과 요령

1. **여러 변수 로깅**: 중요한 변수가 여러 개인 경우, 각각에 대해 Quick Log를 실행하여 여러 줄의 로그를 빠르게 추가할 수 있습니다.

2. **객체 속성 로깅**: 객체의 특정 속성을 로깅하려면, 해당 속성 표현식(예: `user.name`)을 선택하고 Quick Log를 실행합니다.

3. **조건부 로깅**: 조건문 내에서 변수를 선택하고 Quick Log를 실행하면, 조건부 로그를 쉽게 추가할 수 있습니다.

4. **빠른 삭제**: 디버깅이 완료된 후 로그 문을 쉽게 찾아 제거하려면, 특정 패턴(예: '파일명:라인번호')으로 검색하여 일괄 삭제할 수 있습니다.

## 문제 해결

### 플러그인이 작동하지 않는 경우

1. IDE를 재시작해보세요.
2. 단축키 충돌이 있는지 확인해보세요.
3. 설정에서 플러그인이 활성화되어 있는지 확인해보세요.

### 단축키가 작동하지 않는 경우

1. `Settings/Preferences` > `Keymap`에서 `Quick Log` 액션을 검색하여 단축키가 올바르게 설정되어 있는지 확인하세요.
2. 다른 플러그인이나 IDE 기능과 단축키 충돌이 없는지 확인하세요.
